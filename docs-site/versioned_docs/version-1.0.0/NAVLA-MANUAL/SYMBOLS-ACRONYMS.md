| DMR | Deadline Miss Rate | % | Real-time metric |
| AJ | Average Jitter | ms | Real-time metric |

| IK | Inverse Kinematics | — | Solve joint targets from task-space |

| V&V | Verification & Validation | — | Claims → tests → evidence |
