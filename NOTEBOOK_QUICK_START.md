## üêõ Troubleshooting

### Python Not Executing
- First run may take time (Pyodide loading)
- Check browser console for errors
- Ensure code is valid Python

### Packages Not Found
- Some packages may need explicit installation
- Use `!pip install package_name` in a cell

### Output Not Showing
- Check cell executed successfully
- Look for errors in output
- Try re-executing the cell
